# Лабораторная работа 3 - Применение условных вероятностей в анализе данных  
На некотором наборе данных (выбрать самостоятельно, можно: https://www.kaggle.com/datasets) продемонстрировать применение условных вероятностей (см. пример с рейтингом автомобилей по типу кузова, Notebook с примером "Титаника"). Рассматриваемый набор данных необходимо описать (назначение, описание признаков).  

Используемый датасет: "Chocolate Sales.csv" (https://www.kaggle.com/datasets/saidaminsaidaxmadov/chocolate-sales?resource=download) содержит информацию о продажах шоколадной продукции за 2022 год.

Характеристики датасета:
Количество записей: 3,282 транзакции
Количество признаков: 6 основных признаков
Период данных: январь-декабрь 2022 года
